
# Labs Documentation

## Overview
This folder contains practical labs completed as part of my cloud computing and AWS learning journey. Each lab focuses on applying theoretical concepts in real-world scenarios using AWS services and related technologies.

The labs are organized by domain, including Compute, Networking, Security, Storage, and Python. Each section demonstrates hands-on experience with cloud infrastructure, configuration, and best practices.

---

## Compute

### EC2 Instance Setup
This lab focuses on launching and configuring an Amazon EC2 instance.

Key activities include:
- Creating and configuring an EC2 instance
- Selecting appropriate instance types and Amazon Machine Images (AMIs)
- Configuring security groups and key pairs
- Connecting to the instance via SSH
- Basic instance management (start, stop, terminate)

Outcome:
Gained practical experience in provisioning and managing virtual servers in the cloud.

---

## Networking

### Networking Concepts and Configuration
This lab explores core networking principles within AWS.

Key activities include:
- Understanding Virtual Private Cloud (VPC)
- Creating and configuring subnets
- Setting up route tables and internet gateways
- Managing IP addressing and DNS
- Configuring network access and connectivity

Outcome:
Developed an understanding of how cloud networks are structured and how resources communicate securely.

---

## Security

### Monitoring Infrastructure
This lab focuses on monitoring and securing cloud infrastructure.

Key activities include:
- Using monitoring tools to observe system performance
- Setting up alerts and notifications
- Reviewing logs for system activity
- Understanding shared responsibility in cloud security
- Implementing basic security best practices

Outcome:
Learned how to monitor infrastructure effectively and identify potential security risks.

---

## Storage

### Working with EBS (Elastic Block Store)
This lab focuses on block storage in AWS.

Key activities include:
- Creating and attaching EBS volumes to EC2 instances
- Formatting and mounting storage volumes
- Managing volume performance and size
- Creating snapshots for backup and recovery
- Understanding persistence and durability of storage

Outcome:
Gained hands-on experience with scalable and persistent storage solutions in AWS.

---

## Python

### Automation and Scripting
This lab introduces the use of Python for cloud-related tasks.

Key activities include:
- Writing basic Python scripts
- Automating simple tasks
- Working with files and data
- Understanding how scripting supports cloud operations

Outcome:
Developed foundational scripting skills to support automation in cloud environments.
