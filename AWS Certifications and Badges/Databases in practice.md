# Databases in Practice

##  Overview
This module focused on practical database concepts, including how data is stored, managed, and accessed in real-world applications. It provided hands-on understanding of database systems and their role in modern software and cloud environments.

---

##  Key Topics Covered
- Relational vs Non-relational databases
- Database design and normalization
- SQL queries (SELECT, INSERT, UPDATE, DELETE)
- Indexing and performance optimization
- Data integrity and constraints

---

##  Tools & Technologies
- SQL (Structured Query Language)
- MySQL / PostgreSQL (or whichever you used)
- AWS database services (e.g., RDS, DynamoDB if applicable)

---

##  What I Learned
- How to design efficient database schemas  
- Writing and optimizing SQL queries  
- Understanding how databases support applications  
- Managing structured and unstructured data  

---

##  Reflection
This course strengthened my understanding of how databases function behind the scenes in applications. I now feel more confident working with data, designing schemas, and querying databases efficiently. These skills are essential for both backend development and cloud-based systems.
