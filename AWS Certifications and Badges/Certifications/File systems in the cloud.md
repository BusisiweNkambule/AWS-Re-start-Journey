#  File Systems in the Cloud

##  Overview
This module explored how file systems operate in cloud environments, focusing on scalable storage solutions, data access, and management. It highlighted how cloud-based file systems differ from traditional on-premise storage.

---

##  Key Topics Covered
- Types of cloud storage (object, block, and file storage)
- Distributed file systems
- Data durability and availability
- Storage scalability in the cloud
- Access control and permissions

---

##  Tools & Technologies
- Amazon S3
- Amazon EFS (Elastic File System)
- Amazon EBS (Elastic Block Store)
- Cloud storage architectures

---

##  What I Learned
- Differences between object, block, and file storage  
- How cloud file systems scale automatically  
- Best practices for storing and retrieving data in the cloud  
- How to choose the right storage solution for different use cases  

---

##  Reflection
This module helped me understand how modern applications manage large amounts of data using cloud-based storage systems. I learned how to select appropriate storage services based on performance, cost, and scalability needs, which is essential for cloud engineering.
