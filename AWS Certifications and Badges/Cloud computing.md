#  Cloud Computing

##  Overview
This module introduced the fundamental concepts of cloud computing, including how cloud services are delivered, managed, and scaled. It provided a strong foundation for understanding modern cloud infrastructure and its advantages over traditional IT systems.

- Service models (IaaS, PaaS, SaaS)
- Deployment models (Public, Private, Hybrid)
- Scalability and elasticity
- Pay-as-you-go pricing model

- Cloud computing architecture
