# AWS re/Start Program Badge

## Overview

This repository highlights my completion of the AWS re/Start Program, a skills development and job readiness program designed to prepare individuals for entry-level roles in cloud computing.

The program provided hands-on experience with core cloud concepts, AWS services, and real-world scenarios in cloud environments.

---

## Badge Details

- **Program:** AWS re/Start
- **Provider:** Amazon Web Services
- **Focus Areas:** Cloud Computing, Linux, Networking, Security, and Python
- **Level:** Entry-Level Cloud Practitioner / Junior Cloud Engineer

---

## Skills Gained

### Cloud Fundamentals
- Understanding cloud computing concepts (IaaS, PaaS, SaaS)
- Shared Responsibility Model
- Global infrastructure (Regions, Availability Zones)

### Core AWS Services
- Compute: EC2
- Storage: S3, EBS
- Databases: RDS
- Networking: VPC, Subnets, Route Tables
- Identity: IAM

### Networking
- IP addressing and CIDR blocks
- DNS and domain resolution
- Virtual Private Cloud (VPC) configuration

### Security
- Identity and Access Management (IAM)
- Security groups and network ACLs
- Basic cloud security best practices

### Operating Systems & Tools
- Linux command line fundamentals
- Bash scripting basics
- Troubleshooting system performance

### Programming
- Python fundamentals
- Writing simple scripts for automation

---

## Hands-On Experience

During the program, I completed labs and projects involving:

- Launching and configuring EC2 instances
- Hosting static websites using S3
- Creating and managing IAM users and roles
- Setting up VPCs and configuring networking
- Working with databases using RDS
- Monitoring resources using CloudWatch
- Implementing basic automation with Python
